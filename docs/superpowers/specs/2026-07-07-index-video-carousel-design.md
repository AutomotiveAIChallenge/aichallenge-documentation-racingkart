# トップページ 動画回転プレビュー（自動回転カバーフロー）設計

## 目的

トップページ（`index.ja.md` / `index.en.md`）に散在している YouTube 埋め込みを、
1つの「自動回転カバーフロー（coverflow）」プレビューに集約する。放置でゆっくり
自動回転し、矢印・ドット・カードクリックで動画を選択、中央のカードをクリックすると
その場で本物のプレイヤー（iframe）を読み込んで再生する。

## アーキテクチャ

3ファイル構成。**動画リストは JS に単一定義**（日英同一セットのため）。Markdown 側は
空のコンテナ `<div>` を置くだけにし、URL 変更は JS 1ファイルの編集で完結させる。

### 1. `docs/javascripts/video-carousel.js`（新規）
- 動画配列を保持: `{ id?, ja, en }`（`id` が無ければプレースホルダ）
- `document.getElementById('video-carousel')` を見つけて DOM を構築
- キャプションはページ言語（`document.documentElement.lang`）で ja/en を出し分け
- レイアウト計算: 中央からのオフセットで `translateX / translateZ / rotateY / scale` を付与
  （傾きは控えめ、横スライド感を強調。承認済みデモ v2 の値を踏襲）
- 自動回転: `setInterval`（約3.4秒）。カルーセルに `mouseenter`/`focusin` で停止、
  離れると再開。動画再生を開始したら自動回転は止める。
- 中央カードのクリック → そのカードの中身を lazy に `<iframe>`（`autoplay=1`）へ差し替え
- 操作 UI: 前へ/次へ矢印、下部ドット、カードクリック、左右キー
- 二重初期化ガード。Material の instant navigation 対応: `window.document$`
  があれば購読、無ければ `DOMContentLoaded`
- `prefers-reduced-motion: reduce` の場合は自動回転を無効化

### 2. `docs/stylesheets/custom.css`（追記）
- `.video-carousel` 一式のスタイル（既存のブランド配色 `#15151e` / `#4536ac` に合わせる）
- 実在動画: YouTube サムネイル `https://img.youtube.com/vi/<id>/hqdefault.jpg`
- プレースホルダ: グラデーションのタイル＋「準備中」タグ＋ラベル
- レスポンシブ: 画面幅に応じてカード幅・オフセットを縮小（スマホ対応）

### 3. `docs/index.ja.md` / `docs/index.en.md`（編集）
- 冒頭見出し直後に `<div id="video-carousel" class="video-carousel"></div>` を配置
- 既存のインライン `<iframe>` を各セクションから撤去（挑戦課題・セットアップ等）
- 見出し・本文テキストは残す

### 4. `mkdocs.yaml`（編集）
- `extra_javascript` に `javascripts/video-carousel.js` を追加

## データフロー

```
index.md の <div id="video-carousel">
  → video-carousel.js が起動時に検出
    → 動画配列から各カード(サムネ or プレースホルダ)を生成
    → レイアウト適用 → 自動回転開始
      → ユーザー操作(矢印/ドット/クリック/キー)で active 更新・再レイアウト
        → 中央カードのクリックで iframe を lazy 生成し再生
```

## エラー / エッジケース

- サムネ画像の読み込み失敗 → `onerror` でプレースホルダ調のフォールバック背景
- JS 無効環境 → カルーセルは空になるため、コンテナ内に `<noscript>` で
  主要動画への直リンクを最低限用意（任意・簡易）
- プレースホルダのクリック → 再生せず（カーソルは通常、タグで「準備中」を明示）
- instant navigation で戻ってきた際の二重生成防止（`data-initialized` フラグ）

## テスト / 検証

- `mkdocs build` がエラー/警告なく通ること
- `mkdocs serve` で ja/en 両ページを表示し、ブラウザで以下を目視確認:
  - 9枚が表示され自動回転する / マウスで停止・再開
  - 矢印・ドット・カードクリック・左右キーで選択が動く
  - 中央カードのクリックで実在動画が再生される（プレースホルダは再生しない）
  - キャプションが言語ごとに切り替わる
  - スマホ幅でレイアウトが破綻しない

## 非目標（YAGNI）

- 外部カルーセルライブラリの導入（依存を増やさずバニラ実装）
- 動画の全文検索・カテゴリフィルタ
- 言語ごとの動画セット出し分け
