# SW部門

## SW部門概要

SW部門では予選から決勝まで以下のように進みます。

| 項目 | 日程 | 内容 | 参加者 |
| --- | --- | --- | --- |
| SW予選 | 2026年7月頃〜9月頃 | オンラインのシミュレーション環境で走行 | 全参加者 |
| SW決勝 | 2026年9月下旬 | 決勝会場のシミュレーション環境で走行 | SW予選の上位32チーム（未定） |
| 実機決勝 | 2026年9月下旬 | シティサーキット東京ベイ（CCTB）での実車両走行 | 上位16チーム（未定） |

日程・場所などの詳細については運営 HP・運営からのアナウンスをご確認ください。

## SW部門 予選のルール

### 走行方式

- 運営が用意したオンライン上のシミュレーション環境で走行します。
- 参加者が行うことはコードの提出のみです。提出後は自動的にビルド・走行・順位付けが行われます。
- 走行は4台同時のレース方式で行われます。

!!! info "従来との変更点"
    従来はタイムアタック形式でしたが、今回から4台同時走行での順位による競争形式に変わりました。単に速く走るだけでなく、他の車両を追い抜く技術が求められます。

### マッチメイキングとランキング

4台の車両の構成は以下の通りです。

| 車両 | 内容 |
| --- | --- |
| 車両1 | コードを提出したチャレンジャー |
| 車両2 | チャレンジャーのランク帯と近い他の参加者 |
| 車両3 | 運営が用意した NPC |
| 車両4 | 運営が用意した NPC |

レース結果によるランキングの変動は以下の通りです。

- チャレンジャーが勝利した場合は、上位ランキングに移動します。
- タイムアップした場合や NPC が勝利した場合はドローとなり、順位の変動はありません。
- コード提出時以外にも、他の参加者がコードを提出した際に対戦相手として走行する場合があります。この場合に負けると順位も下がります。

### レースの詳細条件

- 周回数は6周で、1位の車両が6周走行した時点で走行終了となります。
- タイムアップは10分です。（★未定）
- 走行開始位置はランダム、または車両番号の順となります。（★未定）

### 速度とペナルティ

- 速度上限は35km/h に設定しています。（★要確認）
- コースの壁への衝突や他車両への自発的な衝突等が発生した場合、ペナルティとして一定時間速度が制限されます。
- 順位によって速度制限などのハンディキャップがつく場合があります。（★未定）

### アイテムシステム

本大会のシミュレータには以下のアイテムが実装されています。詳細は[シミュレータ仕様](../specifications/simulator.ja.md)を参照してください。（★未定）

| アイテム | 効果 |
| --- | --- |
| ニトロブースト | 一時的な加速 |
| 修理アイテム | 車両コンディションの回復 |

### 使用可能なセンサー

- IMU
- GNSS
- Steer Angle
- Wheel Odometry
- V2X 情報（他車両の位置情報）

### 安全ゲート

コード提出時に以下の安全ゲートをすべてクリアする必要があります。（★未定）

- 障害物停止
- NPC 追い越し
- 車線維持

### 禁止事項

以下のような行為・コードは禁止です。

- 他車両の走行を意図的に妨害すること。
- シミュレーション環境そのものをハックすること。（例えば、意図的にCPU負荷や通信負荷を上げるなど）

### 提出回数の上限

1日に提出できるコードの回数には上限があります。

## SW部門 決勝のルール

基本的なルールは予選と同じですが、以下の点が異なります。

- NPCは存在せず、4チームごとのトーナメント戦となる予定です。
- オンライン上ではなく、運営が用意した PC 上での走行となります。
- 走行開始位置は、予選での順位を加味して決められます。
- 1位の車両が6周走行した時点で走行終了となります。（★ これだと2位以下が決まらない）
- クラッシュ等で車両がスタックした場合には、手動操作による復帰が許される可能性があります（★未定）。

## 実機決勝のルール

基本的なルールは予選と同じですが、以下の点が異なります。

### 必須要件

- 実機決勝では、以下の要件を満たしている必要があります。（★ これは不要？ make down と joy での操作で担保？）
  - オペレータの指示により車両を停止させられること。
  - 遠隔での車両操作ができる状態であること。
- 参加者は実車両へのインテグレーション作業を行い、決勝当日に走行に立ち会う必要があります。（作業は運営がサポートします）

### 実機決勝独自のルール

- 走行開始位置は、予選での順位を加味して決められます。
- 1位の車両が6周走行した時点で走行終了となります。（★ これだと2位以下が決まらない）
- クラッシュ等で車両がスタックした場合には、手動操作による復帰が許される可能性があります（未定）。
- 減速ペナルティやニトロなどの加速アイテムはありません。

### 走行中止条件

以下の条件に該当する場合、走行を中止します。

- コースの壁が大きく動いてしまった場合。
- コースから大きく逸脱した場合。
- その他、安全上の問題等でスタッフが指示した場合。
